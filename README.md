# CandyDozzer

> Unity 参考書を元に制作したキャンディ落としゲーム

## 概要

Unity の参考書を参考にしながら制作した 3D ゲームです。
プッシャーを使ってキャンディを落とし、ゲット数を競う「キャンディ落とし」ゲームです。

- プッシャーが前後に自動で動き、キャンディを落とす
- 球形・角形など複数種類のキャンディが登場
- キャンディを落とすとスコア（Candy 数）が加算される
- 射出音やパーティクルエフェクトなど演出も実装
- デスクトップアプリ版（`DesktopAPP`）も同梱

## 使用技術

- Unity（C#）
- Unity Physics（Rigidbody / PhysicMaterial）
- Unity Audio（AudioSource）
- Unity ParticleSystem

## 動作環境 / 注意事項

- Unity でプロジェクトを開いて実行してください
- 動作確認済みの Unity バージョンは不明です
